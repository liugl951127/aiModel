## 总结

本项目共 **20 个视图** + **22 个路由**, 涵盖:
- 1 个登录页 (含 dev 模式/审计/帮助)
- 1 个控制台 (4 stat + 3 列 + 实时事件)
- 9 个业务管理页 (用户/角色/菜单/租户/审计/模型/工作流/版本/案例)
- 6 个 AI 功能页 (智能体/对话/工具/知识库/训练/推理)
- 3 个高级功能 (流程编排/分布式能力/数据流水线)
- 2 个辅助弹窗 (使用说明/案例库)

所有页面共用 MainLayout (浅色侧栏 + 玻璃模糊顶栏 + 实时事件条),
遵循 Element Plus 主题 + 全局 CSS 变量 (light/dark 自动适配).
