## 四、P0 整改清单 (本轮)

### P0-PM-1: 路由角色守卫
**文件**: `frontend/src/router/index.js`
**问题**: 用户能直接输 URL 访问 /system/users (越权)
**修法**: router.beforeEach 检查 `to.meta.roles` 跟 localStorage.roles

### P0-PM-2: 全局 Loading 进度条
**文件**: `frontend/src/main.js`
**问题**: 路由切换无进度, 用户不知道在等
**修法**: 引入 NProgress (内置, 不引新包), router.beforeEach/afterEach 触发

### P0-PM-3: 全局错误边界
**文件**: `frontend/src/App.vue` + `frontend/src/main.js`
**问题**: 组件崩了整页空白
**修法**: Vue 3 `app.config.errorHandler` + 顶层 errorCaptured + 兜底组件

### P0-LEAD-1: 操作审计 (后端)
**文件**: 后端 `audit` 模块新建
**问题**: 改/删/创建无审计
**修法**: AOP + annotation @Audited, 落 audit_log 表

### P0-LEAD-2: 用户协议 + 隐私政策页
**文件**: `frontend/src/views/Agreement.vue`
**问题**: 法务依据缺失
**修法**: 静态页 + 登录前勾选 (强制)

### P0-LEAD-3: 责任边界 + 免责 + SLA
**文件**: `frontend/src/views/Disclaimer.vue`
**问题**: 出事谁负责没说清
**修法**: 静态页 + 顶栏链接
